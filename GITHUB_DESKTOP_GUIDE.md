# GitHub Desktop 推送指南

## 步骤 1：下载安装
1. 访问 https://desktop.github.com/
2. 下载并安装 GitHub Desktop
3. 登录你的 GitHub 账号

## 步骤 2：添加本地仓库
1. 打开 GitHub Desktop
2. File → Add local repository
3. 选择路径：`C:\obsidian_workshop\Roundabout`
4. 点击 "Add Repository"

## 步骤 3：发布到 GitHub
1. 你会看到 "Publish repository" 按钮
2. 点击它
3. 确认仓库名：`obsidian_vault`
4. 选择 "Public" 或 "Private"
5. 点击 "Publish Repository"

## 完成！
现在你的 Obsidian Vault 已经推送到 GitHub 了。

## 后续同步
以后每次修改：
1. GitHub Desktop 会显示更改的文件
2. 填写提交信息（如："更新笔记"）
3. 点击 "Commit to main"
4. 点击 "Push origin"
