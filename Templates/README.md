# 模板库

这里存放常用的笔记模板。

## 📋 可用模板

### 日志类
- [[日报模板]] - 日常记录模板
- [[周报模板]] - 周总结模板

### 工作类
- [[会议纪要模板]] - 会议记录
- [[项目计划模板]] - 项目规划

### 学习类
- [[读书笔记模板]] - 阅读笔记
- [[学习笔记模板]] - 课程/教程笔记

## 🎯 使用方法

1. 复制所需模板
2. 粘贴到相应文件夹
3. 重命名为具体名称
4. 填写内容

---

## 🔧 快速创建

在 Obsidian 中使用 Templater 插件可以自动填充模板变量。

**快捷键**：
- 插入模板：`Alt+E`
- 新建日记：`Ctrl+Alt+D`
