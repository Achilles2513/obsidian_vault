
📁 Projects（项目）      → 当前正在做的事
📁 Areas（领域）         → 长期维护的责任（健康、财务）
📁 Resources（资源）     → 兴趣主题、参考资料
📁 Archive（归档）       → 已完成/不再活跃
📄 Inbox（收件箱）       → 临时想法，定期整理

⚠️ 常见坑
1. 过度分类 → 用链接代替文件夹
2. 完美主义 → 笔记可以烂，但要快
3. 插件依赖 → 定期检查：这个插件我真的在用吗？
4. 忽视回顾 → 每周固定时间浏览图谱，发现连接


**Obsidian 是第二大脑，不是档案馆。每周至少打开旧笔记一次，让知识产生连接，而不是堆积。**
